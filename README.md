# Kami

A fork of Mihon, free and open source manga reader for Android.

I build this fork for my own use. I am not a developer. I have no idea about coding. I only add features that I like from other fork and I only use LLMs (***Claude & Gemini***) to help me write other features that I like. Some of the codes need to adjust for my needs, but for now it is good enough.
